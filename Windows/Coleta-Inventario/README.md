\# Windows Inventory Collector



Script PowerShell para coleta manual de inventário em estações Windows.



\## Objetivo



Executar o script a partir de um pendrive em cada máquina e gerar um arquivo CSV único com os dados de inventário.



\## Dados coletados



\- Nome do computador

\- Usuário logado

\- Sistema operacional

\- Versão do Microsoft Office

\- Processador

\- Memória RAM

\- Tipo de disco: HD, SSD ou desconhecido

\- Tamanho do disco

\- Modelo do equipamento

\- Serial number

\- Asset Tag do BIOS

\- Número de inventário informado manualmente



\## Como usar



Copie os arquivos para um pendrive:



```text

Coleta-Inventario.ps1

Executar-Coleta.bat
